**注意事项：**

+ 关节零点应设置在机械臂直立的默认位置，否则示教模式和主臂模式下的重力补偿功能可能无法正常运行。
+ 系统采用状态机进行模式管理，示教模式、轨迹模式、主从臂（联动控制）模式以及 Web 控制模式相互独立。例如，在轨迹控制模式下若需要调整速度，需先通过 Web 按钮切换至 Web 控制模式；切换后轨迹运动会停止，方可进行速度设置。
+ 关节自动设置零点时，电机会先运动至机械限位位置，再回到正确位置。在此过程中请勿对机械臂进行阻挡或干预。
+ 关节 6 在手动或自动设置零点后，系统都会执行一次校准过程，请勿阻挡该过程。
+ 连接主从臂时，应确保主臂与从臂处于相近位置，否则从臂会快速运动以跟随主臂位置，存在安全风险。
+ 使用夹爪主从控制时，需要在自定义控制器栏中选择“夹爪”类型，并将夹爪使能后，方可正常使用。
+ 进行运动学控制时，应尽量避免关节 J2、J3、J4、J5 处于 0 位，以降低机械臂进入奇异位形的风险。
+ 如果需要在<font style="color:#DF2A3F;">nero上使用以前购买的piper夹爪（新购买的不需要)</font>，在第一次使用是需要按<font style="color:#DF2A3F;">《nero夹爪id更改指南》</font>对夹爪id进行更改后方可接入nero使用，否则会造成夹爪id冲突问题，对运动学，重力补偿都会有严重影响。
+ 机械臂侧装定义为在默认零点位置下，将网口方向朝下进行安装。倒装定义为将机械臂完全倒置。

### 2.1.1控制面板说明
机械臂底座面板上为用户提供一个航空插头，一个标准网口，面板上另外包含WIFI天线和状态指示灯。航插包含两组线路，一组电源为机械臂供电，一组can为用户提供上层和机械臂的通讯；一个内网接口，用于技术支持读取机械臂数据。

注意：内网接口<font style="color:#DF2A3F;">非对外接口</font>，正常使用过程中<font style="color:#DF2A3F;">无需插接任何线束</font>，其作用为松灵技术支持在提供技术服务过程中获取机械臂数据使用，请勿在未获取松灵技术支持同意的情况下使用该接口。

+ 2、使用4颗M3*8内六角沉头螺栓将法兰盘固定至挡板两侧
+ 3、将爪夹、示教器插入法兰盘中（需留意防呆接口）

## 6.1 运动控制

| 运动方式 | 使用流程 | 备注 |
| --- | --- | --- |
| 点位运动 | 输入机械臂末端点位（x,y,z,rx,ry,rz）,点击应用，机械臂会移动到指定点位；<br/>也可以通过点击x y z的图标去增加或减少xyz坐标值，以改变末端的坐标，**点击应用，机械臂会移动到指定点位。** |      ** 在原点位置，不能下发点位控制，因为原点是奇异点，需要先用关节控制j2和j4移动一下，再下发控制！！！！**<br/>输入的点位可能是无法到达的点位，此时机械臂不会移动，且会有异常的提示。 |
| 圆弧运动 | 输入三个点位：起点（x,y,z,rx,ry,rz）,中点（x,y,z,rx,ry,rz），终点（x,y,z,rx,ry,rz）机械臂会根据三个点位，规划并进行圆弧运动<br/> | 输入的点位或圆弧运动规划出来轨迹中的点位可能会出现无法到达的点位，此时机械臂不会移动，且会有异常的提示。 |
| 关节运动 | 输入每个关节的关节角度，点击应用，机械臂底层就会响应执行。<br/>此外还可以通过拖动滑杆，改变数值大小，**注意还是需要“点击应用”才会生效** | 下发的关节角如果超过关节限位会不移动，需要在关节限位内下发控制。 |
| 直线运动 | 输入机械臂位置姿态（x,y,z,rx,ry,rz）,机械臂会直线移动到指定点位； | 输入的点或者直线规划出的轨迹中的点位之间如果有不能到达的点位，运动状态会有无解的情况 |

## 6.4 联动控制
这个界面的核心功能是设置和切换机械臂的工作模式

`普通`：这是独立工作模式。选择此模式时，机械臂（或该控制单元）将不参与任何主从联动，作为一个独立的个体运行。。

`主臂`：这是主导控制模式。指定当前控制的机械臂作为“主机”。在主从联动中，主臂负责主导运动，并用于驱动“从臂”。

`从臂`：这是跟随控制模式。指定当前控制的机械臂作为“从机”。从臂将实时跟随主臂的运动指令，实现精确的同步运动。

<img src="https://cdn.nlark.com/yuque/0/2026/png/33709157/1768443614573-81eb68dc-2645-4524-b800-f98350b3c453.png" width="387" title="" crop="0,0,1,1" id="PZOry" class="ne-image">

**注意：使用主从联动控制时，必须将主臂和从臂都回到零点附近，再使用此功能。因为将2个臂设置为主从臂之后，从臂会迅速同步到主臂的位姿，此时可能会有意外情况发生。**

**       不要同时将两条机械臂同时设为从臂模式，并且不要在两条机械臂连接时同时开启CAN推送。否则会引起外网CAN通讯异常，出现这种情况，需要重启设备。**

## 6.6 轨迹
轨迹管理是集创建、管理、执行和存储机器人运动轨迹的核心控制界面，用户可以通过它便捷地规划一条由多个“运动点”组成的连续路径，并对其进行保存、调用和批量处理。

**包含按钮**：

+ `**新增轨迹**`：创建一个全新的、空白的轨迹。
+ `**刷新**`：刷新下方轨迹列表，获取最新数据。

+ `**导入**`：从本地导入已有的轨迹文件，快速加载使用。

## 6.7 设置零点
       机械臂长时间运行，由于机械磨损等多种因素影响，机械臂的零位可能会出现偏差，当机械臂关节零位出现偏差后，我们需要在操控区零点设置里面重新校准关节零位。分为手动设置零点和自动设置零点。

**注意：关节零点应设置在机械臂直立的默认位置，否则示教模式和主臂模式下的重力补偿功能可能无法正常运行。设置零点后，机械臂关节出现移动后，请勿阻挡。**

### 6.7.1自动设置零点
       自动设置零点可以选择某一个关节，勾选自动设置，点击确认。

**注意：关节自动设置零点时，电机会先运动至机械限位位置，再回到正确位置。在此过程中，请勿对机械臂进行阻挡或干预！！！**

### 6.7.2手动设置零点
手动设置零点可以选择某一个关节，把关节失能，然后手动转动到零位，点击确认。

# 10 机械臂CAN协议
机械臂针对用户的开发提供了CAN接口， 用户可通过该接口对机械臂进行指令控制。

机械臂中CAN通信标准采用的是CAN2.0B标准， 通讯波特率为1000K， 报文格式采用MOTOROLA(MSB)格式。

如需使用CAN指令开发，请联系松灵技术支持获取CAN协议手册。

# 11 维护与保养
为了使机械臂能够长期保持较高的性能，必须进行维护与保养。维护和保养操作务必严格遵守本手册的所有说明。维护和保养的目的是为了确保机械臂及其配件正常运转。

## 11.1 存放需知
使用完毕需复位零点后断电存放，线束做收纳整理，避免出现线束缠绕或者大力牵拉。

工作及存储环境要求：温度：-20-50℃，湿度：25%-85%，非冷凝。

## 11.2 保养需知
维护保养需在机械臂零点断电情况下进行，断电后进行如下操作：

:::warning
**<font style="color:#DF2A3F;">注意：</font>**  
机械臂清理：使用湿巾或者干抹布蘸水后对机械臂进行灰尘、污垢、油渍的清理。

螺丝紧固检查：检查螺丝是否有松动。

电机检查：检查电机及其结构件是否有破损或形变。

线缆检查：检查电源线、CAN线是否稳固，表皮是否有破损

:::

## 11.3 紧急情况处理
在紧急情况下，可直接插拔航插断电处理，需注意断电后机械臂会失能直接掉落，注意避免机械臂损坏或损坏周围环境及人员。拔线处理是应急性处理手段，并不用于防止伤害。
